AI Emotion Detector: Multimodal Analysis
This repository contains a Multimodal Emotion Detection system designed to run seamlessly on Google Colab. By integrating text, image (facial expression), and voice analysis, the system provides a comprehensive emotional profile using state-of-the-art transformer models and signal processing.
Features
•
Multimodal Fusion: Combines three distinct data streams for robust emotion recognition.
•
Gradio UI: A clean, web-based interface for uploading files or recording live data.
•
Google Drive Integration: Automatically saves logs, processed outputs, and session data to your Drive.
•
Colab Optimized: Leverages GPU acceleration for real-time inference.

Setup and Installation
Since this project is built for Google Colab, no local installation is required. Follow these steps:
1.
Open the Notebook: Upload the .ipynb file to your Google Colab environment.
2.
Mount Google Drive:
The script will prompt you to authorize Google Drive access to store results:
Python
from google.colab import drive
drive.mount('/content/drive')
3.
Install Dependencies:
Run the first cell to install the necessary libraries:
Bash
pip install gradio transformers librosa torch torchvision soundfile
How It Works
1. Text Analysis
Uses Natural Language Processing (NLP) to analyze the sentiment and specific emotional tone of input strings.
2. Image Analysis
Processes static images or frames to detect facial landmarks and classify expressions into categories like Happy, Sad, Angry, Neutral, etc.
3. Voice Analysis
Extracts MFCC (Mel-frequency cepstral coefficients) and other acoustic features from audio files to identify emotional prosody based on the RAVDESS standard.
4. Gradio Interface
The UI provides three input components:
•
Textbox for text input.
•
Image Upload for facial recognition.
•
Audio Mic/Upload for voice emotion detection.
